# Predicción de Precios de Viviendas - Proyecto ML

Este proyecto utiliza Machine Learning para estimar el valor de casas basado en su calidad y dimensiones.

## Estructura
- `backend/`: Lógica del modelo y predicción.
- `data/`: Dataset de entrenamiento (`train.csv`).
- `models/`: Archivo del modelo entrenado (`.pkl`).
- `app.py`: Interfaz de usuario por consola.
